# AI Coding Agent Security Requirements

This folder is reserved for the **AI Coding Agent Security Requirements** tool.

The tool is based on the document **The New SDLC With Vibe Coding** and is intended to translate the document's concepts into practical security requirements for organizations using AI coding agents.
